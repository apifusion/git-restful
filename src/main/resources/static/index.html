git-restful documentation

Endpoints

  • git-restful - API revision
  • lock-repo?url=gitRepoUrl - reads git repository
  • list/{repo}/branches - gives branches
  • lock-branch/{repo}/{branch} - reads branch
  • projects/** - work directory browsing: reads source project directory and files.
  • pages/** - reads files. Used for server side forward by /projects on files.
  • docs/{repo}/** - generate and list documentation for source folder/file
  • docgenerator/* - scripts for documentation generation from sources
